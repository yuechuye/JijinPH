# 基金周涨幅榜 + 动量评分

每周计算精选场内 ETF 的周涨幅排名与动量得分，辅助基金轮动买入决策。

## 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行脚本
python cli.py update
```

## 工作原理

### 阶段一：周涨幅排名（周五→周四窗口）

```
上周五净值 → 本周四净值
周涨幅 = (周四净值 / 周五净值 - 1) × 100%
```

按主题分组，每个主题内按周涨幅降序排列，取 Top N。

### 阶段二：动量评分

使用多周期收益率加权，过滤短期噪音：

```
动量得分 = 1周回报 × 40%
         + 2周回报 × 30%
         + 4周回报 × 20%
         + 12周回报 × 10%
```

所有时间点均取周四净值，确保与排名窗口对齐。缺失周期权重按比例重新分配，可用周期不足 2 个的不参与评分。

### 数据源

- [东方财富基金净值接口](https://api.fund.eastmoney.com)（通过 akshare `fund_etf_fund_info_em` 获取）
- 每只 ETF 每 0.2 秒请求一次（带内置 rate limiting）

## 运行时机

**建议：每周四收盘后（15:00 之后）运行**

这样数据最新鲜——周四跑完脚本，周五一早看结果，周五开盘/盘中执行买入。间隔仅一晚，排名信号不被周末消息面污染。

### 前端页面

```bash
# 启动本地服务器查看
python3 -m http.server 8080
# 打开 http://localhost:8080
```

三个 Tab：
- **🏆 总榜** — 所有 ETF 按周涨幅混排 Top 10
- **🚀 动量榜** — 跨主题按动量得分排名的 Top 20
- **各主题 Tab** — 每个板块内部按周涨幅排名

## 推荐买入策略

1. **周四收盘后** 运行 `python cli.py update`
2. 查看动量榜，关注动量得分最高的 2-3 个板块
3. 在对应主题 Tab 中选该板块内周涨幅靠前的 ETF
4. **周五开盘/盘中** 执行买入
5. **每周轮动**：跌出动量榜前 5 的卖出，新上榜的买入

## 配置

编辑 `config/themes.yaml` 可自定义：
- `topN`：每个主题展示的基金数量
- `themes[].funds`：每个主题包含的 ETF 代码列表

## 文件结构
